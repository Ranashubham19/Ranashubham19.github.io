Face Detection in Machine Learning

This project involved creating a system capable of detecting human faces in images or video streams in real-time.

Key Technologies Used: Python, OpenCV (for image loading, preprocessing, and drawing bounding boxes), TensorFlow/Keras (for building and utilizing deep learning models like Haar Cascades, HOG, or more advanced CNN-based detectors like MTCNN or SSD). NumPy was used for efficient array manipulation.

Core Functionality: The system processes input images/frames, applies a chosen face detection algorithm (e.g., Haar Cascades for speed, or a deep learning model for higher accuracy), identifies potential face regions, and draws bounding boxes around the detected faces.

Challenges & Learning: Challenges included handling variations in lighting, pose, occlusion, and scale. This project provided valuable experience in computer vision fundamentals, deep learning model implementation, and performance optimization techniques for real-time applications.

Online Examination System

This project aimed to build a robust and user-friendly web application for conducting online examinations.

Key Technologies Used: The system was developed using PHP for server-side logic, MySQL for database management, and HTML, CSS, JavaScript, and Bootstrap for the front-end user interface.

Core Features:

  • Secure user authentication (students and administrators).
  • Dynamic question fetching from the database.
  • Support for various question types (e.g., multiple-choice, true/false).
  • Timed test environment.
  • Automatic submission and real-time grading upon completion.
  • Result display and potentially basic analytics for administrators.
  • User-friendly interface designed for ease of navigation and accessibility.

Challenges & Learning: Key challenges involved ensuring security against cheating, managing session state effectively, handling concurrent users, and designing an intuitive UI. This project provided practical experience in full-stack web development, database design, and building secure applications.
